# 의류쇼핑몰 OT
## 소개
상품을 구매하기 쉬운 UI를 구성하여 사용자로 하여금 상품구매에 편의성을 제공, 서양의 대표 스트릿 브랜드들 처럼 보기만해도 스타일리쉬한 옷이 한국에도 공급이 필요하다 생각하여 개발하게 된 프로젝트 입니다.
## 기술
* java, html5, css, javascript, ajax
* spring, mybatis, maven, node.js
* oracle, mongoDB
* github
